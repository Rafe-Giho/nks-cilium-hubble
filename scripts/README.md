반복 작업 자동화를 위한 스크립트를 관리합니다.

## 현재 스크립트

현재 재사용 가능한 스크립트는 없습니다.

삭제한 Cilium full replacement 실험 스크립트는 `docs/91-full-replacement-assets-archive.md`에 요약 보존합니다.

## 작성 규칙

- 가능한 한 멱등적으로 작성합니다.
- 입력값은 인자나 환경변수로 받습니다.
- 비밀정보를 하드코딩하지 않습니다.
- 사용법과 전제 조건을 파일 상단 주석 또는 별도 문서로 남깁니다.

## 추가 시점

- 수동 절차가 2회 이상 반복되거나 검증 자동화가 필요할 때 추가합니다.
